# 隐私说明

最后更新：2026-07-16

## 项目角色

Hhh Prompt Studio 是在用户浏览器中运行的开源扩展。本项目当前不运营用于转发分析请求的后端服务。用户自行选择并配置第三方视觉模型服务商。

## 本地保存的数据

- 模型服务商地址、API Key、模型、行为偏好和主题设置保存在 `chrome.storage.local`。
- 自定义提示词模板保存在扩展本地存储中。
- 最近 50 条成功分析历史保存在扩展 Origin 下的 IndexedDB 中。
- 可选的每日调用计数保存在本机，用于本地模型路由，不等同于服务商账单。

这些数据不会由本项目主动上传到项目维护者的服务器。Chrome 同步是否涉及扩展数据取决于实际使用的存储类型和浏览器行为；本项目当前使用本地存储，而非 `chrome.storage.sync`。

## 分析时发送的数据

用户开始分析后，扩展会把以下数据直接发送到用户配置的模型接口：

- 当前提示词模板生成的请求文本。
- 所选图片的 Data URL、源图片 URL，或当前可见标签页截图，取决于图片传输配置和入口。
- 模型标识以及 Chat Completions 请求所需参数。

模型服务商如何记录、处理、保留或用于训练这些数据，由该服务商的条款和隐私政策决定。处理敏感材料前，用户应自行确认服务商政策。

## 权限用途

- `contextMenus`：提供图片右键分析入口。
- `storage`：保存本地配置、Key 和偏好。
- `activeTab` 与 `scripting`：在用户主动操作的当前标签页打开工作台或选图。
- `downloads`：导出分析历史。
- 可选网站访问权限：仅在用户配置模型 Origin 或启用悬停分析时申请。

生产 Manifest 默认不声明永久 `<all_urls>` 权限。

## 导出与删除

团队配置导出不包含 API Key，但可能包含用户自定义提示词。用户可以在扩展界面删除历史与模板。卸载扩展通常会移除其本地数据；建议先导出需要保留的内容。

## 日志与问题反馈

诊断设计不会主动输出 Authorization Header、完整 API Key、完整 Data URL 或完整私人提示词。用户提交 Issue 前仍应检查并移除所有秘密信息和敏感图片。

安全问题请按 [SECURITY.md](SECURITY.md) 中的方式私下报告。
