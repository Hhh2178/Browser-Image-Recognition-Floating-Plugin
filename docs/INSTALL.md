# 安装说明

## 开发安装

1. 运行 npm install。
2. 运行 npm run build。
3. 打开 chrome://extensions。
4. 启用“开发者模式”。
5. 点击“加载已解压的扩展程序”。
6. 选择本项目的 .output/chrome-mv3 目录。

## 更新

重新构建后，在扩展管理页点击该扩展的刷新按钮。配置、模板和历史保存在扩展本地存储中。

## 默认入口

- 网页图片或缩略图链接右键菜单“Hhh：悬浮窗分析图片”。
- 图片会直接在悬浮窗中打开；无法定位时才进入手动选图。
- 模型设置、行为权限和提示词管理都在悬浮窗内打开，不会跳转新标签页。
- 浏览器工具栏图标。
- Ctrl+Shift+Y 截图分析。
- 设置页中可选启用网页图片悬停分析。

Chrome 内部页面、扩展商店和其他禁止注入的页面无法打开工作台。
