# Systems

This folder documents runtime systems and project operating systems. Each system document should describe purpose, ownership, runtime shape, interfaces, permissions, failure modes, and verification.

## Current Systems

- `extension/`: Chrome extension runtime, content/background split, permissions, storage, analysis flow.
- `harness/`: project governance checks, release checks, and documentation contracts.
- `workbench/frontend-design/`: visual and functional standards for the floating workbench.
